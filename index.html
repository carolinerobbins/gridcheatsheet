Grid Positioning

Property Definition
grid-template-columns Defines the line names and track sizing functions of the grid columns. The number of width values determines the number of columns and each width value can be either in pixels(px) or percentages(%).
grid-template-rows Defines the line names and track sizing functions of the grid rows. The number of width values determines the number of columns and each width value can be either in pixels(px) or percentages(%).
grid-gap shorthand way of setting the two properties grid-row-gap and grid-column-gap. It is used to determine the size of the gap between each row and each column. The first value sets the size of the gap between rows and while the second value sets the size of the gap between columns.
grid-row shorthand for the grid-row-start and grid-row-end properties specifying a grid itemâ€™s size and location within the grid row. The starting and ending row values are separated by a /. There is a corresponding grid-column property shorthand that implements the same behavior for columns.
grid-row-start defines on which row-line the item will start.The keyword span can be used to automatically calculate the ending value from the starting value or vice versa.
grid-row-end defines how many rows an item will span, or on which row-line the item will end. The keyword span can be used to automatically calculate the ending value from the starting value or vice versa.
